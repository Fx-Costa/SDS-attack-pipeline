# SDS-attack-pipeline
A Python pipeline tool for analysis and attacks (data poisoning) on the SDS (k-anonymity/DP) synthesis

## Prerequisites
- lib-python (SDS)
- python-pipeline (SDS)
- core (SDS)
- rustc compiler (https://www.rust-lang.org/)
- maturin
